## Descripcion

Dashboard web para visualizacion y gestion de proyectos Fizzy con metricas de desempeno del equipo.

## Funcionalidades

- Vista general de proyectos y su estado
- Tracking de tareas por proyecto y por persona
- Metricas de productividad y desempeno individual
- Reportes de avance en tiempo real
- Integracion con API de Fizzy
